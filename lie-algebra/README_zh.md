# Julia中的李代数泛包络代数

## 导师

王至宏 (zhihong@stu.ecnu.edu.cn)

## 项目描述

项目旨在使用 Julia 编程语言实现李代数的泛包络代数，关键特性包括：
- 对给定的李代数，生成其泛包络代数
- 支持代数上的基本运算
- 与社区现有的计算机代数系统（CAS）工作兼容

## 要求

- 对计算机代数系统和符号计算感兴趣
- 具备一定的李代数知识，了解单李代数的分类定理
- 熟悉 Julia 编程语言
